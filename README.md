# Python Random Module Demo

## Description
This repository demonstrates basic usage of the built-in `random` module in Python. The examples show how to generate random numbers, select random items from a list, and simulate simple random behavior. The goal is to provide clear and beginner-friendly examples.

## Installation
No installation is required. The `random` module is part of Python’s standard library.

Make sure Python is installed on your system.

## How to Run
Run each file using Python:

```bash
python random_integer.py
python random_choice.py
python dice_simulator.py
python random_float.py
```
## Files and Examples

- `random_integer.py`
Generates a random integer between 1 and 10.

- `random_choice.py`
Selects a random item from a list.

- `dice_simulator.py`
Simulates rolling a six-sided die.

- `random_float.py`
Generates a random float between 0 and 1.

## Purpose
These examples show how to use the random module in simple programs. This module is useful in many situations, such as simulations, games, and testing.
